The mapme design system.

This document captures what's live in the product today — logo, colors, type, components, patterns. It's a reference, not a constraint. Iterate as the product evolves and update this page when you do.

Logo

Hexagonal earth, a river through it.

The mapme mark is a layered cube. Green on top represents Planet — soil and growth. Purple below represents People — the humans and communities living on and with the land. The wave that cuts through is the Mekong — the river that shaped the whole story.

Anatomy

Four stacked strata in an isometric cube, with a river wave flowing across. Each layer is a value, not just a color.

  • Lime Upper stratum — surface life, seasonal
  • Olive Lower green — rooted memory, soil
  • Light purple Front face — community, the lived layer
  • Deep purple Back face — structural, institutional
  • Wave River — flow of water, people, materials

Typography

Fraunces for voice. Manrope for everything else.

Fraunces (variable serif) carries mapme's editorial voice — display, headings, quotes, long-form prose. Manrope (sans) handles interface — navigation, buttons, labels, body UI. Fraunces is configured with Optical Size 144, Soft 60, Wonk 1 per the brand spec.

Components

The primitives.

Buttons, chips, badges, inputs, cards, verification badges, map pins. Everything below is live in the product. Reuse these before building new variants.

Buttons

Four weights. Primary for the one big action on a page. Secondary for alternates. Outline for tertiary. Ghost for destructive or low-emphasis. Artisan size (56px, 16px text) for elder-user flows.

Chips (filter / interest)

Round, 9×14px padding. Multi-select by default. Active state carries accent color.

Badges — status

Live / pending / error / new. Uppercase 10.5px. Use sparingly — one per card maximum.

Four levels from unverified to master recognition. Used on place profiles and artisan cards.

Card

Primary content container. 14px radius, 1px border, 18–24px padding. Title uses Fraunces, body uses Manrope.

Accessibility

Built-in, not bolted on.

mapme is used by elder artisans on low-end devices in rural connectivity. Accessibility isn't a compliance checkbox — it's the product working.

Contrast

All text passes WCAG AA against its background. Purple accent (#a370f7) on bg-2 (#16161a) passes AAA for body text.

Tap targets

44px minimum throughout. 56px for artisan flows (Chú Vũ's signup, Bác Hùng's dashboard, any path serving elder users).

Type size

16px Body 1 as the default. 18px+ for artisan paths. Layouts must survive 200% browser zoom without horizontal scroll.

Voice input

Every long-form text field in artisan flows carries a voice-record companion. Auto-transcribed. The story doesn't need keyboards.

Offline-first

Form data persists if connection drops. Signup wizards resume at the last step. Map tiles cached where possible.

Screen readers

All icons carry aria-label. SVG logos carry role="img" + title. Map pins carry accessible name when keyboard-focused.

Bilingual standard

Vietnamese first, English alongside.

mapme is a Vietnamese product with international collaborators. Content is authored in Vietnamese. English translations accompany for navigation, titles, and key CTAs.

Default language

Vietnamese is the default for place owners and artisans (Chú Vũ). English-alongside for explorers (Linh). Auto-detect via browser locale, override with toggle.

What gets translated

Navigation, page titles, section headings, primary CTAs, card titles, metric labels. Long-form prose stays in the language it was authored.

Implementation

Every translatable string carries data-vi + data-en attributes. A header toggle swaps via JavaScript. Form data is preserved across language changes.

Curator voice is bilingual

When Thy Dang writes about nghĩa trang, her Vietnamese words stay. The English version is an editorial re-translation, not a machine output.

Do / Don't

Guardrails that matter.

Opinions held strongly, earned through product use. Break them only with a clear reason.

✓ Do

Use purple as the primary interactive accent. Every button, link, and focus state should anchor to the same color.

✗ Don't

Use rainbow. Layer colors are for differentiation of Community Mode layers, not decoration.

✓ Do

Use Fraunces for editorial voice — headlines, pull quotes, essay body. The serif is mapme's tone.

✗ Don't

Use Fraunces for UI labels, buttons, or form fields. Manrope handles interface.

✓ Do

Credit curators by full name + role, prominently. Thy Dang's Cemetery is her work, not mapme's.

✗ Don't

Collapse curator attribution into "mapme editorial." Name people; protect their voice.

✓ Do

Show pending/unverified states openly. Trust is earned by transparency.

✗ Don't

Hide unverified content. A "Chưa xác minh" badge is better than silently gatekeeping.

✓ Do

Test every flow with Bác Hùng — can a 65-year-old artisan use this meaningfully? If yes, ship it.

✗ Don't

Assume tech fluency. Voice input, big buttons, Vietnamese-first aren't edge cases; they're the target.
